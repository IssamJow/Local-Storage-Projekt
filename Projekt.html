HTML LocalStorage Tutorial

Mit diesem Tutorial soll gezeigt werden, wie Local Storage überhaupt funktioniert und angewendet wird.
Außerdem werden die Varianten des Local Storage von einander unterschieden und in praktischen Beispielen demonstiert.
Zu guter Letzt werden die Vorteile geneüber Cookies dargestellt und gezeigt , wieso Local Storage eine bessere Anwendung für Webstorage ist.

Local Storage Definition

Mit Local Storage könne Webanwendungen local Daten im Browser des Benutzers speichern

Vor HTML5 mussten Anwendungsdaten in Cookies gespeichert werden, die in jeder Serverabfrage drin waren.Localer Speciher ist sicherer und große Datenmengen können local gespeichert werden, ohne die Performence der Webseite zu beeinflussen

im Gegensatz zu Cookies ist die Speichergrenze viel größer (mind.5MB) und Daten werden niemals an der Server übertragen.

Local Storage Varianten

HTML local storage bietet 2 Varianten für die Speicherung von Daten auf dem Client an:

  • window.localStorage - speichert Daten ohne verfallsdatum
  • window.sessionStorage - speichert Daten für eine Sitzung(Daten gehen verloren, wenn die Registerkarte im Browser geschloßen wird)

Bevor man local storage anwenden kann , sollte man prüfen ob der Browser localstorage-fähig ist:

if(typeof(Storage) !== "undefined") {
// Code for localStorage/sessionStorage.
} else {
// Sorry! No Web Storage support..
}

The localStorage

Speichert die Daten ohne Ablaufsdatum.Die Daten werden nach dem Schließen des Browsers nicht gelöscht, sondern sind für den nächsten Tag , Woche oder Jahr wieder verfügbar.

Beispiel für localStorage:

// Store
localStorage.setItem( "studiengang" , "MedienInformatik");
// Retrieve
document.getElementById("result").innerHTML = localStorage.getItem( "studiengang");

Beispiel erklärt :

  • Erstellt ein name/value Paar mit dem Name="studiengang" und dem Wert="Medieninformatik".
  • ruft den Wert von "studiengang" auf , und fügt es in den Element mit dem id="result".

Das obere Beispiel könnte auch so aussehen:

// Store
localStorage.studiengang = "MedienInformatik;
// Retrieve
document.getElementById("result").innerHTML = localStorage.studiengang;

The syntax for removing the "studiengang" localStorage item is as follows:

// Loescht bestimmte Items.
localStorage.removeItem("studiengang");

Oder alle ....

// Loescht alle Items des localStorage.
localStorage.clear();

The following example counts the number of times a user has clicked a button. In this code the value string is converted to a number to be able to increase the counter:

Beispiel

if (localStorage.clickcount) {
localStorage.clickcount = Number(localStorage.clickcount) + 1;
} else {
localStorage.clickcount = 1;
}
document.getElementById("result").innerHTML = "You have clicked the button " +
localStorage.clickcount +" time(s).";

The sessionStorage Objekt

The sessionStorage object is equal to the localStorage object, except that it stores the data for only one session. The data is deleted when the user closes the specific browser tab.

The following example counts the number of times a user has clicked a button, in the current session:

Beispiel für sessionStorage:

// Store
sessionStorage.setItem( "studiengang" , "MedienInformatik");
// Retrieve
document.getElementById("result").innerHTML = sessionStorage.getItem( "studiengang");
